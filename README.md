## 4. Verify Essential Services

### 4.1 SSH

Command:

```
sudo systemctl status ssh
```

Expected output:

```
ssh.service - OpenBSD Secure Shell server
Loaded: loaded (/usr/lib/systemd/system/ssh.service; disabled; preset: enabled)
Active: active (running)
```

If SSH is not active:

```
sudo apt install openssh-server -y
sudo systemctl enable ssh
sudo systemctl start ssh
```

## 5. Test Network Connectivity

### 5.1 Check IP Address

Command:

```
ip a
```

Interpretation:

- Network interface is active
- IP address assigned (example: 10.0.2.15/24)
- IPv6 present, loopback OK

### 5.2 Internet Connectivity Test

Initial command:

```
ping -c 4 8.8.8.8
```

Common issue: ping is not installed by default on Ubuntu Server 24.04 LTS

Solution:

```
sudo apt install iputils-ping -y
```

Test after installation:

```
ping -c 4 8.8.8.8
```

Expected output:

